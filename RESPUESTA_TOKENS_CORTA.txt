## 🎯 Respuesta Directa

**SÍ**, el login guarda el token y se usa automáticamente en TODAS las APIs.

---

## ⚡ RESUMEN EN 3 PASOS

### 1️⃣ LOGIN GUARDA TOKEN
```javascript
// LoginView.vue
const res = await authService.login(email, password);
// Backend responde con accessToken

authService.setTokens(accessToken, refreshToken);
// ✅ Guardado en localStorage
```

### 2️⃣ INTERCEPTOR LO LEE
```javascript
// AxiosConfig.js - Interceptor de Request
axios.interceptors.request.use((config) => {
  const token = localStorage.getItem('accessToken');
  if (token) {
    config.headers.Authorization = `Bearer ${token}`;
    // ✅ Agregado a TODOS los requests
  }
  return config;
});
```

### 3️⃣ APIs USAN EL TOKEN AUTOMÁTICAMENTE
```javascript
// products.js, orders.js, etc.
async getAll() {
  return await axiosConfig.doGet('/api/products');
  // ✅ El interceptor ya agregó el token
  // ✅ No necesitas hacer nada más
}
```

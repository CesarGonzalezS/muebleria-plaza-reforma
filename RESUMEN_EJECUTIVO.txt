## 🚀 Cómo Usar

### Paso 1: Asegurar que el backend está corriendo
```
http://localhost:8080
```

### Paso 2: Ejecutar el frontend
```bash
npm run dev
```

### Paso 3: Abrir en navegador
```
http://localhost:5173
```

### Paso 4: Registrarse o iniciar sesión
- Ir a `/register` o `/login`
- Se guardan los tokens automáticamente

### Paso 5: Explorar las pantallas
- Ir a `/api-test` para ver todos los links
- O navegar manualmente

## 🎨 Características

### Frontend
- Vue 3 + TypeScript
- Vue Router con lazy loading
- Axios con interceptors
- SweetAlert2 para mensajes
- Bootstrap Icons
- Responsive design
- Validaciones en formularios

### Backend Integration
- JWT Tokens (24 horas)
- Autenticación automática
- Manejo de errores centralizado
- Protección de rutas por rol
- CORS configurado

### Seguridad
- Tokens guardados en localStorage
- Headers Authorization automáticos
- Validación de permisos (USER vs ADMIN)
- Redireccionamiento en caso de expiración

## 🔗 Endpoints Mapeados

### Autenticación
- POST /api/auth/register
- POST /api/auth/login
- POST /api/auth/change-password
- POST /api/auth/forgot-password
- POST /api/auth/reset-password
- GET /api/auth/validate-reset-token

### Productos
- GET /api/products
- POST/PUT/DELETE /api/products/{id}
- GET /api/products/low-stock

### Inventario
- GET /api/inventory/movements/{productId}
- POST /api/inventory/adjust
- POST /api/inventory/add-stock
- POST /api/inventory/remove-stock

### Órdenes
- GET/POST /api/orders
- GET /api/orders/{id}
- PATCH /api/orders/{id}/status

### Reportes
- GET /api/reports/sales
- GET /api/reports/products
- GET /api/reports/low-stock-alerts

### Clientes (Nuevo)
- GET/POST/PUT /api/customers
- GET /api/customers/{id}/stats
